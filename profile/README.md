# HomogeneousTools

HomogeneousTools is a toolset to work with homogeneous varieties and homogeneous vector bundles on them,
mostly with a view towards sheaf cohomology.

It is still a work-in-progress, with contributions by

* Pieter Belmans
* Max Briest
* Javier Fernandez Piriz

If you want a sneak preview, please get in touch with Pieter
